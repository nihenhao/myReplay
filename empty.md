# 切忌出现空白页面

用户绝不期望看到空白的，毫无信息的页面，这会让用户产生迷惑及不信任感。用户会不知道发生了什么事，以及他们能继续做什么。

####例1：如果页面中确实没有可显示的信息，要给予提示

如图，若用户从未在数据模型中增加过自定义表，则该页面的列表确实为空。
